# GrowthScoreTreasure (成长积分宝)

✨ 一款专为家长设计的轻量级网页端小程序，用于记录孩子日常行为的积分奖惩，通过「奖励加分+过失扣分」的方式，直观追踪孩子的成长轨迹。

## 核心功能
- 📝 积分记录：支持自定义行为类型（如“主动整理玩具+5分”“挑食偏食-2分”），快速新增/编辑/删除积分记录；
- 📊 数据可视化：以日历/折线图展示积分趋势，清晰看到孩子的行为变化；
- 👶 多娃适配：支持添加多个孩子的积分账户，独立管理不混淆；
- 💻 网页端适配：无需下载APP，浏览器直接访问，PC/平板/手机多端兼容。

## 适用场景
家长日常记录孩子的行为积分，通过正向激励+适度约束，帮助孩子养成良好习惯，同时留存成长过程中的点滴细节。

## 技术栈（可根据你的实际技术栈修改）
前端：HTML + CSS + JavaScript / Vue3 / React
数据存储：LocalStorage（本地）/ Firebase（云端）
UI：TailwindCSS（轻量化）

## 开发目标
打造简单、无广告、轻量化的积分记录工具，聚焦亲子场景的核心需求，让家长轻松记录孩子的成长。
